## Staff Roles at a Glance

| Role | What You Can Do | Start Page |
|------|----------------|------------|
| **Cashier** | Take orders, process payments, clock in/out | New Sale |
| **Kitchen / Checkoff** | View kitchen tickets, log production, record waste | Kitchen Display |
| **Warehouse** | Manage inventory, purchase orders, stock counts | Inventory |
| **Accountant** | View reports, manage expenses, invoices | Finance Dashboard |
| **Admin / Manager** | Everything above + settings, products, users | Dashboard |
| **Executive / Owner** | Full unrestricted access | Dashboard |
| **Custom Roles** | Tailored permissions across 30+ categories | As configured |

> The sidebar only shows sections your role can access.
